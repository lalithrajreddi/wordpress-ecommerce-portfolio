# WordPress E-commerce Website

This repository documents an e-commerce website that I have been developing and maintaining since 2021 using WordPress.

## Project Overview
The website is a fully functional e-commerce platform built on WordPress, focusing on product management, content updates, and performance maintenance. The project involves regular updates, customization, and hands-on management of a live website.

## Responsibilities
- WordPress installation and setup
- Theme customization
- Plugin configuration (e-commerce, security, SEO)
- Product and content management
- Website maintenance and updates

## Technologies & Tools
- WordPress
- WooCommerce
- HTML, CSS
- Basic PHP customization
- Hosting & domain management

## Notes
This repository does not include WordPress core files. It serves as a portfolio and documentation of real-world WordPress development and maintenance experience.
